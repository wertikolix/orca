# Orca 0.2.2

Release focus: root renderer layout flexibility for easier integration into existing scroll containers.

## Highlights

- Added `OrcaRootLayout` configuration for both render entry points:
  - `OrcaRootLayout.LAZY_COLUMN` (default)
  - `OrcaRootLayout.COLUMN`
- New `rootLayout` parameter on:
  - `Orca(markdown = ...)`
  - `Orca(document = ...)`
- This removes the hard requirement to have `LazyColumn` at the root and simplifies embedding Orca into screens that already manage scrolling.
- Updated README with usage examples and behavior notes.
