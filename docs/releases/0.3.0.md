# Orca 0.3.0

Release focus: footnotes support in parser and Compose renderer.

## Highlights

- Added `commonmark-ext-footnotes` support in `orca-core`.
- Parser now maps:
  - inline footnote references (`[^label]`) to `OrcaInline.FootnoteReference`
  - footnote definitions to `OrcaBlock.Footnotes` with `OrcaFootnoteDefinition`.
- Compose renderer now renders:
  - footnote references as superscript-style markers (`[1]`)
  - footnote definitions as a numbered block section in document flow.
- Added unit tests for:
  - parse mapping of references + definitions
  - inline reference rendering and fallback behavior
  - stable footnote numbering map generation.
