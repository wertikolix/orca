# FhMd 0.1.0-alpha02 (draft)

Release focus: stabilization, documentation, and CI.  
No engine scope expansion compared to `0.1.0-alpha01`.

## Highlights

- Added GitHub Actions CI (`build + tests`) for pushes and pull requests to `main`.
- Updated project docs to make support boundaries explicit.
- Kept parser/renderer scope unchanged to reduce regression risk before beta.

## Supported (unchanged)

- Parser backend: `commonmark-java` (`0.27.1`)
- Public parser API:
  - `interface FhMdParser { fun parse(input: String): FhMdDocument }`
- Blocks:
  - heading
  - paragraph
  - bullet list / ordered list
  - quote
  - fenced code block / indented code block
- Inlines:
  - text
  - bold
  - italic
  - inline code
  - link
  - soft/hard line breaks
- Compose renderer:
  - `FhMd(markdown = ...)`
  - `FhMd(document = ...)`
  - `onLinkClick` callback
  - baseline styling via `FhMdStyle`

## Not Supported Yet (unchanged)

- tables
- images
- HTML blocks/inline HTML
- strikethrough
- task lists
- thematic breaks (`---`)
- footnotes/front matter/extensions
- non-Android targets (multiplatform-ready architecture only)
