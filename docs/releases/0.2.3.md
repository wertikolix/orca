# Orca 0.2.3

Release focus: bare URL linkification parity with previous Markwon behavior.

## Highlights

- Enabled `commonmark-ext-autolink` in `orca-core`.
- Bare URLs like `https://example.com` are now parsed as links and rendered as clickable link annotations in Orca.
- Existing markdown link syntax `[text](url)` behavior is unchanged.
- Added parser and renderer integration tests for bare URL linkification.
