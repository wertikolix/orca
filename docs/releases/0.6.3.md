# Orca 0.6.3

Release focus: stable publishing pipeline.

## Highlights

- `OrcaMarkdownParser` remains the primary parser API.
- `IntellijMarkdownOrcaParser` remains as a deprecated compatibility alias.
- Central publishing reliability improvements:
  - in-memory signing works with empty passphrase values;
  - CI signing key preparation handles escaped and multiline secret formats;
  - per-publication `javadoc` artifacts are included for Central checks.

## Migration from 0.6.0

```diff
- import ru.wertik.orca.core.IntellijMarkdownOrcaParser
+ import ru.wertik.orca.core.OrcaMarkdownParser

- val parser: OrcaParser = IntellijMarkdownOrcaParser()
+ val parser: OrcaParser = OrcaMarkdownParser()
```
