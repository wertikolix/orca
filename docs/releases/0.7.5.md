# 0.7.5

Syntax highlighting expansion release.

## Added

- Syntax highlighting support for **Python**, **Go**, **Rust**, **C**, **C++**, **Swift**, **C#**, **Java**, **TypeScript**
- SQL keyword matching is now case-insensitive (`SELECT`, `FROM`, `WHERE` etc. are highlighted correctly)

## Changed

- Expanded syntax highlighting test coverage for all new languages

## Upgrade

```kotlin
implementation("ru.wertik:orca-core:0.7.5")
implementation("ru.wertik:orca-compose:0.7.5")
```

No API changes. Drop-in replacement for 0.7.4.
