# Orca 0.6.4

Release focus: parser/runtime ergonomics for streaming UIs.

## Highlights

- Added parser-level cache API:
  - `parseCached(key, input)`
  - `parseCachedWithDiagnostics(key, input)`
- Added structured diagnostics API:
  - `parseWithDiagnostics(input): OrcaParseResult`
  - warnings/errors are available through `OrcaParseDiagnostics`.
- Added configurable security policy for renderer:
  - `OrcaSecurityPolicy`
  - `OrcaSecurityPolicies.Default`
  - `OrcaSecurityPolicies.byAllowedSchemes(...)`
- `Orca(markdown = ...)` now supports:
  - `parseCacheKey`
  - `securityPolicy`
  - `onParseDiagnostics`

## Migration from 0.6.3

No breaking changes. Existing `parse(...)` and `Orca(...)` usage remains valid.
