# Orca 0.4.0

Release focus: syntax coverage expansion and renderer usability.

## Highlights

- Added HTML support in parser:
  - `OrcaBlock.HtmlBlock`
  - `OrcaInline.HtmlInline`
  - rendered with safe text fallback in Compose renderer.
- Added front matter parsing in `orca-core`:
  - YAML (`--- ... ---`)
  - TOML (`+++ ... +++`)
  - exposed via `OrcaDocument.frontMatter`.
- Added inline footnotes (`^[...]`) support.
- Added footnote navigation in renderer:
  - tap reference marker (`[n]`) to jump to footnote definition
  - tap backlink (`â†©`) to return to source block.
- Added syntax highlighting for code blocks (enabled by default, configurable).
- Added table auto layout mode by content width (default), with fixed-width mode still available.
